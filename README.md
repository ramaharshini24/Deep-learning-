# Neural Networks and Deep Learning Projects

## Projects

### 1. Neural Network Fundamentals & Optimization
- Implemented L1 and L2 norm computations and visualizations.
- Analyzed convexity properties of norm-based unit balls in R².
- Derived optimization expressions for sigmoid-based artificial neurons.

### 2. Feed-Forward Neural Network for Image Classification
- Developed a neural network from scratch using Python and NumPy.
- Implemented forward propagation and backpropagation algorithms.
- Evaluated Sigmoid, Tanh, and ReLU activation functions.
- Performed dimensionality reduction and feature analysis.
- Visualized training performance using accuracy and loss metrics.

### 3. Recurrent Neural Networks for Sequence Prediction
- Built RNN, LSTM, and GRU models using PyTorch.
- Trained models with Adam optimization and MSE loss.
- Compared recurrent architectures through validation-loss analysis.
- Developed preprocessing pipelines for sequential datasets.
- Visualized learning behavior and model performance.
